# 📊 Nutrition, Physical Activity & Obesity Dashboard (Power BI)

## 🚀 Project Overview
This project focuses on analyzing obesity trends using a real-world government dataset. The dashboard is created using Power BI and provides interactive insights into factors affecting obesity.

---

## 📂 Dataset
- Source: Government Health Dataset
- Contains data on:
  - Year
  - State
  - Category Type (Age, Income, Education, etc.)
  - Category
  - Topic (Obesity, Physical Activity)
  - Value (%)

---

## 🛠 Tools & Technologies
- Power BI
- Power Query (Data Cleaning)
- DAX (Data Analysis Expressions)

---

## 📊 Dashboard Pages

### 🟦 1. Overview Dashboard
- KPI Cards (Avg, Max, Min)
- Map Visualization
- Trend Line Chart
- Donut Chart

### 🟦 2. Demographic Analysis
- Age-wise analysis
- Gender comparison
- Income & Education impact

### 🟦 3. Advanced Insights
- Scatter Plot (Activity vs Obesity)
- Heatmap (State vs Category)
- Top 10 States Analysis
- Category Comparison

### 🟦 4. Smart Dashboard
- Trend Analysis
- Factor Analysis (Stacked Bar / Decomposition Tree)
- KPI Monitoring
- Gauge Chart

---

## 🔍 Key Insights
- Obesity trends are increasing over time
- Income and education significantly influence obesity
- Physical activity has a strong relationship with obesity
- Certain states show higher obesity levels
